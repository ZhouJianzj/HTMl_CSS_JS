CSS的实现方式有三种方式：
1、第一种方式内联式定义方法
	主要的语法结构实现<p style = "样式名字 : 样式的值 ; 样式名字 : 样式值"></p>
2、第二种方式是以样式块的形式实现的
	具体的实现语法是
	<head>
		<style type = "text/css">
			选择器{
				样式名字 : 样式取值;
				样式名字 : 样式取值;
				...
			}
		</style>
	</head>
3、引入的方式取是由css
	具体的实现是 
	<link type = "text/css rel = "stylesheet" href = "样式的路径名字"/>
	优点是维护成本比较低
